# LLM-Chat-Collapser-Sidebar

> Last Updated: 2025-12-11

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://raw.githubusercontent.com/CaOUMay/LLM-Chat-Collapser-Sidebar/refs/heads/main/LICENSE)

A userscript to collapse code blocks and long messages in LLM chat interfaces (e.g. Gemini and ChatGPT), and provide a draggable “question box” — enabling jumps to specific questions.

### Features

* Collapse long code blocks and user messages to keep the chat view clean.  
* Provide a “⚙” menu at the bottom-right to customize the range where collapsing starts, and provide simple appearance modifications.
* Provide a question box (sidebar): resizable, draggable, and collapsible.  
* Supported: **ChatGPT**; the question-box on **Gemini** is currently not supported.  

### Note

* The question-box feature on **Gemini** still has some issues; further modifications are required.  

### Installation

1. Install a userscript manager such as [Tampermonkey](https://www.tampermonkey.net/) or [Violentmonkey](https://violentmonkey.github.io/).  
2. Install this script: **[Install from GitHub](https://raw.githubusercontent.com/CaOUMay/LLM-Chat-Collapser-Sidebar/main/LLM-Chat-Collapser-Sidebar.user.js)**

## LLM 聊天内容折叠器

一个用户脚本，用于折叠 LLM 聊天界面（如 Gemini 和 ChatGPT）中的代码块和长消息，并且提供可拖动的问题框，并实现点击后跳转到指定问题

### 功能特性
* 折叠冗长的代码块和用户提示，保持聊天视图整洁。
* 可通过右下角的「⚙」自定义开始折叠的范围，并提供简单的外观修改
* 提供问题框，可自定大小、位置，问题框可折叠
* 支持：**ChatGPT**，**Gemini** 暂不支持问题框

### 注意
* 在**Gemini**上运行还存在一些问题，待后续修改

1.  安装用户脚本管理器，如 [Tampermonkey](https://www.tampermonkey.net/) 或 [Violentmonkey](https://violentmonkey.github.io/)。
2.  安装此脚本: **[从 GitHub 安装](https://raw.githubusercontent.com/CaOUMay/LLM-Chat-Collapser-Sidebar/main/LLM-Chat-Collapser-Sidebar.user.js)**
